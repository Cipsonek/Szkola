Ciąg Fibonacciego to ciąg liczb naturalnych określony rekurencyjnie.

F(n)=Fn-1+Fn-2 dla n>0

  • F0=0
  • F1=1
  • F2=1
  • F3=2
  • F4=3
  • F5=5
  • F6=8
  • F7=13
  • F8=21
  • F9=34
Fibonacci( n )
if n=0 then return 0
if n=1 then return 1
f' ← 0
f ← 1
for i ← 2 to n
do
m ← f + f'
f' ← f
f ← m
end
return f
